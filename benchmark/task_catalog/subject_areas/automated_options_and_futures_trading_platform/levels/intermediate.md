Create a web dashboard and a backend service to manage automated trading for options and futures contracts. The backend should connect to a brokerage API using this websocket endpoint wss://api.brokerage.com/live to listen to S&P 500 index fund data. If the index drops by more than 2% in an hour, automatically execute a market order to buy 10 put options. Save all trade history to a local database. Set up the repo, build the services, and start the servers.
