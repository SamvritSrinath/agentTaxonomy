# Sakila Duplicate Customer Cleanup

This repository assists in identifying and resolving duplicate customer records in the
`sakila_master.db` SQLite database. **No destructive changes are performed automatically.**
All proposed deletions require explicit human confirmation.

## Prerequisites

- SQLite 3 command‑line tool (`sqlite3`)
- The `sakila_master.db` database file in the repository root
